# 军用与政务公文格式规范（测试参考）

## 1. 公文组成要素
一份正式公文通常由以下要素组成：
- **标题**：应当准确简要地概括公文的主要内容并标明公文种类。
- **主送机关**：公文的主要受理机关，应当使用全称、规范化简称或者同类机关统称。
- **正文**：公文的主体，阐述公文的具体内容。
- **落款（发文机关署名）**：署发文机关全称或者规范化简称。
- **成文日期**：署公文形成的具体日期。

## 2. 常用公文种类定义
- **请示**：适用于向上级机关请求指示、批准。
- **报告**：适用于向上级机关汇报工作、反映情况、回复上级机关的询问。
- **通知**：适用于发布、传达要求下级机关执行和有关单位周知或者执行的事项，任免人员。

## 3. 写作基本要求
- **政治性**：必须符合党的路线方针政策和国家法律法规。
- **准确性**：事实可靠，数据准确，用词贴切。
- **严谨性**：结构严密，逻辑清晰。
- **简洁性**：文字精炼，不拖泥带水。
